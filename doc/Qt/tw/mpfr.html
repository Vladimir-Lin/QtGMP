Windows 32 / 64

Under Windows, MPFR is required to compile with MPIR.

Currently, Only MPFR 3.0.0 can support MPIR under Windows.

  1. Uncompress mpfr-3.0.0.tar.xz
  2. Uncompress mpfr-3.0.0.build.vc10.zip
  3. Go to mpfr-3.0.0/build.vc10
  4. Look for lib_mpfr and Open up, VC is considered a crap and unstable, try multiple times please.
  5. rename mpir-2.6.0 into mpir
  6. copy everything in lib/(x64,Win32)/Release into build.vc10/(x64,Win32)/Release
  7. move build.vc10/(x64,Win32)/Release to build.vc10/lib(x64,Win32)/Release
  8. Go to VC and setup include path to mpir/lib/(x64,Win32)/Release
  9. Compile and collect your files at build.vc10/lib/(x64,Win32)/Release


Android ARMv7

Before you build GNU GMP for Android, you need to create a toolchain root system by issue the following command:

/home/Qt/Android/ndk/build/tools/make-standalone-toolchain.sh --platform=android-15 --toolchain=arm-linux-androideabi-4.8 --install-dir=/home/Builds/androidsys
