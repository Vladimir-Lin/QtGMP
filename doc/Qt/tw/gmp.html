GNU GMP is the foundmental building block of many systems, that includes CIOS Exciton system. Exciton uses GNU GMP all over its modules.


Linux 32 / 64

Linux is most easy one to compile GNU GMP, since GNU GMP is mostly developed under Linux.

./configure --disable-shared --enable-static
make

This will do just fine.

Linux 32 one step compilation script : gmp-linux-x86.sh
Linux 64 one step compilation script : gmp-linux-x64.sh


Mac OS X Clang

./configure --disable-shared --enable-static
make

Mac OS X is easy one.

Mac OS X one step compilation script : gmp-mac.sh


Windows - MPIR

Under Windows, GNU GMP replacement is MPIR. And, to compile MPFR, you are required to compile MPIR at least for once. Normally, you will compile MPIR and MPFR together.

MPIR source site : http://www.mpir.org

The method to build MPIR, please refer to here.


Android ARMv7

Before you build GNU GMP for Android, you need to create a toolchain root system by issue the following command:

/home/Qt/Android/ndk/build/tools/make-standalone-toolchain.sh --platform=android-15 --toolchain=arm-linux-androideabi-4.8 --install-dir=/home/Builds/androidsys

Then,

export TKX=/home/Builds/androidsys
export ACC=arm-linux-androideabi

export CC="$TKX/bin/arm-linux-androideabi-gcc --sysroot=$TKX/sysroot"
export CXX="$TKX/bin/arm-linux-androideabi-g++ --sysroot=$TKX/sysroot"
export AR="$TKX/bin/arm-linux-androideabi-ar"
export RANLIB="$TKX/bin/arm-linux-androideabi-ranlib"
export SYSROOT="$TKX/sysroot"
export PATH="$TKX/bin":$PATH

./configure --host=$ACC --disable-shared --enable-static
make


Then, copy whatever you need.  For compiling MPFR, create directory like gmproot, then please includes in "include", and libgmp* in "lib".

Android one step compilation script : gmp-android.sh


iPhone OS 7.0 ARMv7

To compile GNU GMP for iOS, you are required to find out the current sysroot.

For example,
export IROOT=/Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/Developer/SDKs/iPhoneOS7.0.sdk

and then, find out the root directory of your compiler tool chains
export IOS=/Developer/Platforms/iPhoneOS.platform/Developer

To configure, replace all compile tools with iOS toolchains

./configure CC="$IOS/usr/llvm-gcc-4.2/bin/llvm-gcc-4.2" CXX="$IOS/usr/llvm-gcc-4.2/bin/llvm-g++-4.2" CPP="$IOS/usr/llvm-gcc-4.2/bin/llvm-gcc-4.2 -E" LD="$IOS/usr/bin/ld" CPPFLAGS="-arch armv7 -isysroot $IROOT -miphoneos-version-min=4.2" AR="$IOS/usr/bin/ar" NM="$IOS/usr/bin/nm" NMEDIT="$IOS/usr/bin/nmedit" LIBTOOL="$IOS/usr/bin/libtool" LIPO="$IOS/usr/bin/lipo" OTOOL="$IOS/usr/bin/otool" RANLIB="$IOS/usr/bin/ranlib" STRIP="$IOS/usr/bin/strip" --disable-shared --enable-static --prefix=$IOS/usr/local --host=none-apple-darwin10

make


iPhone OS 7.0 one step compilation script : gmp-ios.sh
